# Pragma IA - Documentación del Proyecto

**Última actualización:** 1 de marzo de 2026
**Versión:** 1.0  
**Autor:** Pragma IA Team

Esta carpeta contiene la documentación esencial del proyecto pragma-ia-solving-with-ai, que centraliza artefactos reutilizables para trabajar con asistentes de IA.

Cada archivo cumple una función específica: detalla diversos aspectos del proyecto, desde la visión general hasta las directrices específicas de implementación.

## Archivos y sus propósitos

### [project-overview.md](project-overview.md)
Contiene la declaración de visión principal, los objetivos principales y una explicación general de lo que el proyecto busca resolver o lograr. Este documento actúa como la guía para todas las demás decisiones.

### [requirements.md](requirements.md)
Desglosa tanto las funciones del sistema (requisitos funcionales como "los usuarios pueden crear instrucciones personalizadas") como su rendimiento (requisitos técnicos como "compatibilidad con GitHub Copilot y Amazon Q").

### [project-structure.md](project-structure.md)
Describe la estructura del sistema: cómo se conectan y comunican las diferentes partes, cómo se ve la estructura de las carpetas y cómo fluyen los artefactos a través del repositorio.

### [tech-stack.md](tech-stack.md)
Justifica las elecciones tecnológicas, explicando por qué se seleccionaron formatos, herramientas y plataformas específicas para las diferentes partes del proyecto.

### [features.md](features.md)
Profundiza en las características individuales del proyecto, describiendo exactamente cómo funciona cada componente, incluyendo casos de uso y reglas de validación específicas.

### [implementation.md](implementation.md)
Abarca cómo se construye y mantiene el proyecto: el enfoque de desarrollo, los estándares de contenido y las directrices técnicas específicas que los contribuyentes deben seguir.

### [user-flow.md](user-flow.md)
Mapea los flujos funcionales de usuarios y datos a través del sistema: una hoja de ruta detallada que muestra cada paso en la creación y uso de artefactos.
